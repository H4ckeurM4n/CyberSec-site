# OSINT

L'enquête à partir de sources ouvertes : transformer des informations publiques
éparpillées en une vision claire d'une cible.

!!! note "Données publiques uniquement"
    L'OSINT se limite à des informations **accessibles publiquement et
    légalement**. Pas de contournement d'accès, pas d'intrusion.
